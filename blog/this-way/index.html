Drawing: This way

2022-05-17

An illustration conveying a child's dream, as strange as those can be. I wanted the owl to be slightly terrifying, but also fearfull of the boy. And part of the environment, as if it's hiding there for a long while.

This time I spent some more time refining the watercolor underlayer; instead of opting for one color I added some nuances in the way the light reveals certain yellows, while keeping the shadows warm and dark.
